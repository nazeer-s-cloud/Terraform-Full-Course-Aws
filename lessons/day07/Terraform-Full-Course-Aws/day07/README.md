# Day 07 - Terraform Variable Type Constraints (Minimal Testing Code)

This folder contains minimal Terraform code to test all variable type constraints:
- string  
- number  
- bool  
- list  
- set  
- map  
- tuple  
- object  

Resources are simulated using **null_resource** to avoid AWS charges.

## How to Run

terraform init
terraform validate
terraform plan


## What This Project Demonstrates
- String prefix for naming
- Number used for dynamic counts
- Booleans toggling behavior
- Lists for ordered values
- Sets for validation
- Maps for tags
- Tuple for structured values
- Object for grouped settings
- Summary output combining mixed types
